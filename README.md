# Anemone Noble Canvas

## Overview

**Anemone Noble Canvas** is a web platform dedicated to unique, hand-painted clothing. The site allows users to explore a range of pre-painted garments, order custom designs, or bring their own clothing for personalized artwork. The user experience is enhanced by a modern navigation bar featuring a collapsible menu, a search function, and a shopping cart button for future e-commerce integration.

## Features

- **Custom Painted Clothing**: Browse and order garments with unique hand-painted designs.
- **Personalized Orders**: Users can select from existing designs or request custom artwork on their own clothing.
- **User Dashboard**: Registered users have access to their profile, order history, and personalized settings.
- **Dynamic Navigation**: Features a collapsible sidebar menu, quick-access logo, search bar, and cart button.
- **Responsive Design**: Layout and styles are optimized for desktops and mobile devices.

## Technology Stack

- **Frontend**: HTML, CSS (with modular styles), JavaScript (vanilla, for interactivity)
- **Navigation/UI**: Dynamic menu, search bar, and cart implemented with custom JS and CSS
- **User Authentication**: Simulated login/logout state (can be extended for real authentication)
- **Image Gallery**: Swiper-style gallery for showcasing products

## File Structure

- `index.html`: Main landing page with navigation, introduction, and product highlights
- `Styles/generalStyles.css`: Core styles for navigation, layout, and interactive components
- `JS/general.js`: Handles menu, search, and basic UI interactions
- `JS/home.js`: Manages login/logout state and user-specific navigation
- Additional pages: `Pages/profile.html`, `Pages/product.html`, `Pages/dashboard.html`, etc.

## Planned Features

- **Shopping Cart Functionality**: Currently styled, with future plans for e-commerce
- **Search Capability**: Expandable search bar for product and content discovery
- **User Profiles**: Enhanced customization and order management

## How to Run

Simply open `index.html` in your browser. All navigation and interactive components work with vanilla JS and CSS.
